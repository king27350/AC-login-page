## Installing 
Clone my project from github
```
$git clone https://github.com/king27350/AC-login-page.git
```
Installing package
```
npm install 
```
---
### Running the tests
Start my project
```
npm run dev
```
Input on new web page
```
http://localhost:3000/
```
End the project
```
ctrl + c 
```
